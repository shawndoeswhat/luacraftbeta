# LuaCraftBeta

## Requirements

- JDK 8
- Maven

## Installation

1. Clone the repository.

2. Install the dependencies.

    ```bash
    ./build.sh
    ```

3. Build the project.

    ```bash
    mvn clean package
    ```

4. Place the output `target/LuaCraftBeta-{version}.jar` into your Poseidon server's `plugins` directory.
